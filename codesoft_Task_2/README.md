# IMDB Movie Rating Prediction

## Overview

This project predicts IMDB movie ratings using machine learning regression techniques.

The dataset contains movie information such as genre, director, actors, duration, and votes. The objective is to identify factors influencing movie ratings and build predictive models.

---

## Dataset Features

* Movie Name
* Year
* Duration
* Genre
* Director
* Actors
* Votes
* Rating (Target Variable)
