# Sales Prediction Using Machine Learning

## Overview

This project predicts product sales based on advertising expenditure across multiple marketing channels.

The objective is to analyze how advertising investments influence sales performance and build a machine learning model capable of estimating future sales.

---

## Dataset Features

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales (Target Variable)

---

## Project Workflow

### Data Exploration

* Dataset inspection
* Missing value analysis
* Statistical summary

### Exploratory Data Analysis

Visualizations performed:

* Feature Distributions
* Scatter Plots
* Correlation Heatmap

### Data Preparation

* Feature Selection
* Train-Test Split

### Model Building

* Linear Regression

### Model Evaluation

Metrics Used:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

## Key Findings

* TV advertising shows the strongest relationship with sales.
* Radio advertising also contributes positively to sales.
* Newspaper advertising demonstrates comparatively weaker influence.
* Linear Regression effectively models the relationship between advertising expenditure and sales.
